# Contributing

このリポジトリに貢献するときは最初にコミットする内容についてIssueでディスカッションしてください。PRはできる限り[README](README.md)にある項単位でお願いします。そこではその項で言及する内容ついて話し合います。リポジトリのオーナー(ouchiminh)からGOサインが出ればPRを作成してouchiminhにレビューを依頼してください。
問題がなければマージされます。
